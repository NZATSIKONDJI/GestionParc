# Projet Appareils

## Description

Ce projet est une application Java qui modélise différents types d'appareils (ordinateurs et imprimantes) et leurs fonctionnalités, avec des options de personnalisation. Il utilise le design pattern Décorateur pour ajouter dynamiquement des fonctionnalités aux appareils et un pattern Fabrique pour créer des instances d'appareils.

## Fonctionnalités

- Représentation d'appareils avec des attributs tels que l'identifiant, le nom, le prix de base et le type.
- Options de personnalisation via les décorateurs pour ajouter des fonctionnalités de sécurité et de performance.
- Fabrique pour créer des appareils de manière dynamique.
- Gestion d'un inventaire d'appareils avec des fonctionnalités pour ajouter, supprimer et afficher les appareils.
